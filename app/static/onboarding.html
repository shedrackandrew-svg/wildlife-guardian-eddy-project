Phone-as-Camera Onboarding

Pair a phone as an external camera in 3 steps.

Dashboard Live Monitor Admin

Step 0: Connect Backend API (One Time)

No backend API set. Frontend-only mode will run.

Step 1: Name the Camera Source

Click Generate Pair Link.

Step 2: Scan QR on Your Phone

Both devices should be on the same network for best performance.

Step 3: Start Streaming

Open the generated link on your phone, allow camera permission, then tap Start Stream.

The main WildGuard dashboard will begin receiving detections from that phone camera source.

Bluetooth pairing on the web is browser-limited; QR/link pairing works reliably over local and internet networks.
